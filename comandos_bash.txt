COMANDOS GIT 
⦁	Cd : Muda a pasta
⦁	 Ls : O que tem na pasta
⦁	Pwd : O caminho absoluto (mostra aonde vc está)
⦁	Ls -a : acessa "arquivos ocultos", Ex: '.'(ponto), '..'(ponto ponto) 
⦁	Comando .. (ponto ponto) é a pasta "mae",pasta acima do desktop
⦁	Comando . (ponto) é a própia pasta
⦁	Mkdir (criar uma pasta) Ex: mkdir (nome da pasta)
